    • head -n [行数a] # 显示开始a行
    • tail -n [行数a] # 显示最后a行
    • tail -n +[行数a] # 从a行以后开始显示

    前 2000 行

    cat file head -n 2000

    1000-2000 行

    从 1000 行开始显示 2000 行
    cat file tail -n +1000 | head -n 2000

    SED

    sed -n ‘1000, 2000p’ filename
